## selected work

*tools built in the gap between knowing how things break and knowing how to make them.*

---

### [tls-proxy](https://github.com/d3f4lt0/tls-proxy)
> *"you can't unsee what's inside the wire."*

mitm tls interception proxy — dynamic certificate authority, per-host leaf issuance, full traffic decryption. sits silently between the client and the server. reads everything. changes what it wants. a surgical instrument for the paranoid or the precise. both, usually.

`go` `tls` `mitm` `x509` `crypto` `traffic-inspection` `tamper`

---

### [http2-inspector](https://github.com/d3f4lt0/http2-inspector)
> *"most people use protocols. some people read them."*

raw rfc 7540 implementation in go — no abstractions, no http client wrapper. drives the full handshake by hand. prints every frame: settings, headers, data, window updates, goaway. timing to the millisecond. the kind of tool you write when documentation isn't enough and you need to see the actual wire.

`go` `http2` `rfc7540` `protocol-analysis` `frames` `alpn`

---

### [quantum-stealth-kernel](https://github.com/d3f4lt0/quantum-stealth-kernel)
> *"some doors don't have handles. you build your own key."*

async dual-layer evasion engine — ja3/tls cryptographic session impersonation fused with cdp browser fingerprint neutralization. engineered for fortified anti-bot perimeters that assume everyone is a bot. the answer to systems that see you before you speak.

`python` `async` `tls` `cdp` `fingerprint-spoofing` `evasion`

---

### [anti-bot-engine](https://github.com/d3f4lt0/anti-bot-engine)
> *"if you know how the trap works, you know how to build one."*

three-layer go middleware — atomic token bucket via lua/redis, burst entropy detection on request interval deviation, silent honeypot trapping. bots are consistent. humans are not. that inconsistency is the only thing that saves you.

`go` `redis` `lua` `middleware` `entropy` `rate-limiting`

---

### [fingerprint-audit](https://github.com/d3f4lt0/fingerprint-audit)
> *"seeing yourself the way the system sees you."*

vanilla js diagnostic suite surfacing the exact browser telemetry akamai, cloudflare, and perimeterx collect — canvas hash entropy, webgl fingerprinting, headless leaks, behavioral signal drift. zero dependencies. runs in your browser. no data leaves. just the mirror.

`javascript` `browser-fingerprinting` `webgl` `canvas` `security`
